# @require

The `@require` keyword is used to import a module.

## Examples

**Import a module with the original name**
```ranty
# Imports as 'test-module'
@require "test-module"
```

**Import a module with an alias**
```ranty
# Imports as 'tm'
@require tm: "test-module"
```
