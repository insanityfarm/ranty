Introduction

Rant 4.0 is a procedural templating language for generating text and structured values. This reference covers the stable language, runtime behavior, module system, CLI, and standard library.

The in-repo documentation was rebuilt using the archived reference site as the starting map and then rewritten to match the shipped 4.0 implementation.

Contents

  • Getting Started for installation, the first program, and the CLI quickstart.
  • Language for syntax, text behavior, functions, accessors, and control flow.
  • Runtime Features for attributes, formatters, modules, determinism, and the CLI.
  • Standard Library for the exported built-ins grouped by category.
  • Diagnostics for compiler messages and runtime error categories.
