Standard Library: Verification functions

is-some


[%is-some: value]

→ bool

Returns @true if value is any non-nothing type.

Paraneters

value ← any
The value to check.

is-between


[%is-between: value; a; b]

→ bool

Returns @true if value is bewteen a and b (both inclusive).

Parameters

value ← any
The value to check.

a ← any
The first bound.

b ← any
The second bound.
