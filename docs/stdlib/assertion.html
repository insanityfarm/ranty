Assertion

Assertion helpers that raise runtime errors on failure.

NameCall FormSummary
assertRaises an assertion error when the condition is false.
assert-notRaises an assertion error when the condition is true.
assert-eqRaises an assertion error when two values differ.
assert-neqRaises an assertion error when two values are equal.
