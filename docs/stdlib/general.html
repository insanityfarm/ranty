General

General-purpose functions, module helpers, and runtime utilities.

NameCall FormSummary
alt[%alt: a; ...rest]Prints the first argument that is not `nothing`.
call[%call: func; args?]Calls a function with a list of argument values.
cat[%cat: ...values]Prints each argument into the current scope.
either[%either: cond; a; b]Prints `a` when the condition is true, otherwise `b`.
len[%len: value]Returns the length of a string, list, map, or range-like value.
type[%type: value]Prints the runtime type name of a value.
seed[%seed]Prints the currently active RNG seed.
tap[%tap: ...]Consumes arguments and produces no output.
print[%print: ...values]Prints values into the caller's output scope.
range[%range: a; b?; step?]Builds a half-open integer range.
require[%require: path]Imports a module through the active module resolver.
irange[%irange: a; b?; step?]Builds an inclusive integer range.
fork[%fork: seed?]Pushes a derived RNG onto the RNG stack.
unfork[%unfork]Pops the most recent derived RNG.
try[%try: context; handler?]Runs a function with an optional runtime-error handler.
ds-request[%ds-request: id; ...args]Calls a registered data source by ID.
ds-query-sources[%ds-query-sources]Returns the list of registered data-source IDs.
proto[%proto: map]Returns a map's prototype.
set-proto[%set-proto: map; proto?]Sets or clears a map's prototype.
