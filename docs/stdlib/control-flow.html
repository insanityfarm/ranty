Attributes And Control Flow

Functions that configure block behavior or expose active block state.

NameCall FormSummary
if[%if: cond]Reference entry for `if`.
elseif[%elseif: cond]Reference entry for `elseif`.
else[%else]Reference entry for `else`.
mksel[%mksel: mode]Creates a selector value for use with `[sel]`.
rep[%rep: reps]Configures repetition count or repetition mode for the current block.
sel[%sel: selector?]Sets the active selector for the current block, or returns it when called without arguments.
sep[%sep: separator]Sets the separator for repeated block iterations.
mut[%mut: mutator?]Assigns a mutator function for block elements.
sel-skip[%sel-skip: selector; n?]Advances a selector without printing the selected value.
sel-freeze[%sel-freeze: selector; frozen?]Freezes or unfreezes a selector in place.
sel-frozen[%sel-frozen: selector]Returns whether a selector is frozen.
reset-attrs[%reset-attrs]Clears the active block-attribute state.
step[%step]Returns the current step number as a one-based integer.
step-index[%step-index]Returns the current step index as a zero-based integer.
step-count[%step-count]Returns the number of steps in the active block.
