Glossary

TermMeaning
BlockA brace-delimited selection unit made of one or more elements.
ElementA single selectable item inside a block.
HintingTreating an expression as text for surrounding whitespace behavior.
SinkingRemoving pending adjacent whitespace around the next text unit.
RepeaterA block running under repetition, usually through `[rep]`.
SelectorAn object that chooses the next block element.
ScopeA local output and variable environment.
