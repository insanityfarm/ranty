Keywords

Rant keywords are prefixed with `@` and provide built-in control-flow and module operations.

  • @require: import a module.
  • @return: return from the nearest function.
  • @continue: continue the nearest repeater iteration.
  • @break: stop the nearest repeater.
  • @weight: assign a block-element weight.
  • @text: mark a definition as text-producing for auto-hinting.
