Functions

Define named functions with `[$name: params] { ... }` and call them with `[name: args]`. Arguments are separated with semicolons.

[$square:x] {[mul:<x>;<x>]}
[square:3]

Parameters

A required parameter is written as `x`. Optional parameters use `x?`. Variadic parameters use `args*` for zero or more values or `args+` for one or more values.

Lambdas

Anonymous functions use `[?: params] { ... }` and capture surrounding variables by reference when needed.

Piping

Use `|>` inside a function-call argument list to pipe a result into the next function call.

[split: "the quick brown fox"; \s |> filter: []; [?:word] { [len:  |> le: 3] } |> join: \s]
