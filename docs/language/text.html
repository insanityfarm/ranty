Text

Plain source text becomes output fragments. By default, same-line whitespace between text units is normalized to a single ASCII space.

One  two   three

Line Breaks

A line break between adjacent text fragments does not insert a space by itself.

Water
melon

Hinting

Use a backtick before an expression when it should participate in surrounding text spacing.

<$name = "world">Hello, `<name>!

Sinking

Use `~` to remove pending adjacent whitespace around the next text-producing unit.

{\:} ~{\(}
