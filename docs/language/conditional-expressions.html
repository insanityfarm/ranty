Conditional Expressions

Conditional expressions are built from `@if`, `@elseif`, and `@else`. Conditions are evaluated from left to right until the first truthy branch succeeds.

@if 0: {
  zero
} @else: {
  nonzero
}

Truthiness

TypeTruthiness
boolUnchanged.
intTruthy when nonzero.
floatTruthy when nonzero and not NaN. Infinity is truthy.
string, list, map, rangeTruthy when non-empty.
tuple, function, selectorAlways truthy.
nothingAlways falsy.

Once a branch succeeds, later conditions are not evaluated and their bodies are not run.
