Accessors

Accessors read and write variables, collection members, and nested values.

Getters

Use `` to read a variable, `` to index a list, and `` to access a keyed map value.

Setters

Setter forms live inside angle brackets as well and support assignment, definition, and compound assignment.

Fallback Values

Use `?` inside an accessor when a missing lookup should fall back to a default value instead of raising an error.

<f/-2 ? 0>

Globals And Descoping

Accessors can target globals explicitly or walk outward through parent scopes when a local binding is shadowed.

Anonymous Accessors

Anonymous accessors let an inline value become the root of a lookup path.
