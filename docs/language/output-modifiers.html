Output Modifiers

Output modifiers are block elements that transform the caller's current output before the element's own result is written back.

The @edit Operator

`@edit` consumes the caller's current output, optionally binds it to a local name, and replaces it with the modifier body.

"example" { @edit x: `<x> `<x> }
"example" { @edit: "overwritten" }

Placement Rules

`@edit` must appear at the start of a block element.
