Data Types

Rant programs operate on a small set of runtime value types. Collections and functions are first-class values, so a script can generate structured output as well as text.

TypeDescriptionTruthiness
stringText data.Truthy when non-empty.
int64-bit signed integer.Truthy when nonzero.
float64-bit floating-point value.Truthy when nonzero and not NaN.
boolBoolean value.Uses its own value.
listOrdered mutable collection.Truthy when non-empty.
tupleOrdered immutable collection.Always truthy.
mapKeyed mutable collection.Truthy when non-empty.
rangeLazy integer range.Truthy when non-empty.
functionCallable value.Always truthy.
selectorSelection strategy object.Always truthy.
nothingUnit-like empty value.Always falsy.
