Attributes

Attributes configure how the next block executes. In stable 4.0 they are exposed through standard-library functions.

AttributePurpose
[rep]Configures repetition count or repetition mode.
[sep]Sets the separator between repeated iterations.
[sel]Sets or returns the active selector.
[mut]Sets a mutator function for block elements.
[if], [elseif], [else]Configure conditional block behavior.
[step], [step-index], [step-count]Expose the current block position.
[reset-attrs]Clears the active attribute state.
