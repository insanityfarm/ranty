Getting Started

Rant can be used through the CLI or embedded as a Rust library.

Install The CLI

cargo install rant --version 4.0.0 --features cli

From a checkout, you can also run the CLI with `cargo run --features cli -- --help`.

Your First Program

[$greet:name] {
  Hello, <name>!
}

[greet:world]

CLI Quickstart

# Start the REPL
rant

# Run inline code
rant --eval '[rep:3][sep:\s]{beep}'

# Run a file
rant examples/helloworld.rant

Embed In Rust

use rant::Rant;

let mut rant = Rant::new();
let program = rant.compile_quiet("Hello, world!")?;
let output = rant.run(&program)?;
